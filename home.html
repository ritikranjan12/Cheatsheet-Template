    Primitive Type Variables


    The eight primitives defined in Java are int, byte, short, long, float, double, boolean, and char those aren't considered objects and represent raw values.

    5. byte is a primitive data type it only takes up 8 bits of memory

    viewsCount = 3_123_456L;
  6. float
    7. We represent basic fractional numbers in Java using the float type. This is a single-precision decimal number. Which means if we get past six decimal points, this number becomes less precise and more of an estimate. 
    price = 100INR;
  7. char
    8. Char is a 16-bit integer representing a Unicode-encoded character. 
    letter = 'A';
  8. boolean
    9. The simplest primitive data type is boolean. It can contain only two values: true or false. It stores its value in a single bit.

  9. int
    10. int holds a wide range of non-fractional number values. 
    var1 = 256;
  10. short
